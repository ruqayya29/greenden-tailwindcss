Greenden is a responsive plant e-commerce website built using HTML and Tailwind CSS. The site is designed for plant lovers who want to buy natural and artificial plants online.
Pages:
1. Home Page

Sticky responsive Navbar with hamburger menu on mobile
Hero section with a Shop Now button
About Us section highlighting key features
Best Seller section showcasing top plants with prices
Footer with contact info and quick links

2. Products Page

Displays the full collection of available plants
Each product shown with image, name and price
Clean grid layout for easy browsing

3. Contact Page

Contact form for customer inquiries
Store location, email and phone number
Easy way for customers to reach out

Tech Used:

HTML5
Tailwind CSS
Responsive Design

Fully responsive on mobile and desktop
